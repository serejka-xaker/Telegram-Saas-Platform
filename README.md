# Telegram SaaS Platform

> SaaS-платформа для создания и управления множеством дочерних Telegram-ботов из единой админ-панели.

##  О проекте

Это SaaS-платформа, которая позволяет создавать и управлять множеством дочерних Telegram-ботов из единой админ-панели. Каждый бот работает в изолированном процессе с собственной конфигурацией, токеном и базой данных, что обеспечивает безопасность и масштабируемость.

###  Ключевые особенности
-  Мультибот-архитектура: запуск неограниченного количества ботов
-  Изоляция процессов: каждый бот работает в отдельном контексте
-  Гибкая конфигурация: индивидуальные настройки для каждого бота
-  Раздельные базы данных: безопасность и независимость данных
-  Единая админ-панель: централизованное управление всеми ботами

## Требования

Перед началом работы убедитесь, что на вашей системе установлены:
- **Docker** и **Docker Compose**

## Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/serejka-xaker/Telegram-Saas-Platform.git
cd Telegram-Saas-Platform
```
### 2. Скопируйте шаблон переменных окружения
```bash
cp .env.example .env
```
### 3. Заполните файл .env необходимыми значениями
```env
TOKEN="Токен вашего Telegram-бота"
POSTGRES_USER=имя_пользователя базы_данных
POSTGRES_PASSWORD=пароль пользователя базы данных
POSTGRES_DB=konstruktor_db
POSTGRES_HOST=konstruktor_postgres
POSTGRES_PORT=5432
ADMIN_TG_ID=Telegram ID администратора
ERROR_CHANNEL_CHAT_ID = Telegram ID чата с ошибками
```
### 4. Запустите проект через Docker Compose
```bash
docker-compose up --build
```
### 5. Проверьте логи приложения
```bash
docker-compose logs -f app
```
